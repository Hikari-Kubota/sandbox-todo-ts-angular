# sample-todo-ts-angular

my sandbox.

## Installation
1. [Vagrant](https://www.vagrantup.com/) と [VirtualBox](https://www.virtualbox.org/) をインストール
2. Vagrant を起動する  
```shell
$ vagrant up
```
3. 起動したら Vagrant に接続する
```shell
$ vagrant ssh
```
4. いろいろインストール
```shell
$ cd /vagrant
$ npm install
```

## Usage

詳しくは package.json がある階層で `npm run` を参照。

### Build
```shell
$ npm run build
```

### Watch
```shell
$ npm run watch
```
